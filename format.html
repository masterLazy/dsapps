格式清理工具

去除文本中的连续换行符，一键清理从网页或 Markdown 复制时产生的多余空行

text_fields 字符: 0 format_line_spacing 行数: 0 space_bar 空行: 0

功能介绍

  • compress 压缩连续换行 — 将连续 2 个及以上的换行替换为 1 个，保留正常段落分隔
  • format_clear 移除所有换行 — 移除所有换行符但保留空格，文字连为一段
  • short_text 压缩全部空白 — 将所有空白字符（换行/空格/缩进）压缩为单个空格
